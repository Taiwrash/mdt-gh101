2. ZK Proof Generator

How to submit this proof:

Option A: Directly on GitHub.com (Recommended)
  1. Click the copy icon above to copy the JSON proof to your clipboard.
  2. Fork this repository on GitHub to your account.
  3. In your fork, navigate to the proofs/ directory and click Add file > Create new file.
  4. Name the file username.json, paste the JSON content, and click Commit changes... (commit directly to `main`).
  5. Navigate back to the original repository and open a Pull Request from your fork.
Option B: Local Command Line
  1. Click the download icon above to save the file as username.json.
  2. Fork and clone the repository.
  3. Create a branch: git checkout -b proof/username
  4. Add your downloaded JSON file to the proofs/ folder.
  5. Commit & Push: git commit -am "add proof", git push origin proof/username
  6. Open a **Pull Request** to the upstream repository!
